Text box mode practice

Use this page to practice how KeyPilot behaves when a text field is focused.

When you focus a text field (like an input or textarea), you need to type normal characters. But KeyPilot’s browsing controls also use characters (like F, D, V…), so KeyPilot temporarily switches into text box mode.

In text box mode:

  • Typing works normally in the focused text field.
  • Most KeyPilot browsing shortcuts are effectively suspended until you exit.
  • Press Esc to exit text box mode and return to normal browsing.

If you move the mouse and hover another element, KeyPilot gives you a short window (about 3 seconds) where you can still click links/buttons with F.
