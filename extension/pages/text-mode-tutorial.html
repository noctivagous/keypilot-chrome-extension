Text box mode

When you focus a text field (like an input or textarea), you need to type normal characters. But KeyPilot’s browsing controls also use characters (like F, D, V…), so KeyPilot temporarily switches into text box mode.

What changes in text box mode?

  • Typing works normally in the focused text field.
  • Most KeyPilot browsing shortcuts are effectively suspended until you exit.

How to exit

Press Esc to exit text box mode and return to normal KeyPilot browsing.

Still need to click something?

If you move the mouse and hover another element, KeyPilot gives you a short window where you can still activate links and other elements.

  • Hover an element, then press F to click it.
  • There’s a brief countdown (about 3 seconds) after mouse movement.

Close this tutorial with Esc (or E) while the popover is open.
